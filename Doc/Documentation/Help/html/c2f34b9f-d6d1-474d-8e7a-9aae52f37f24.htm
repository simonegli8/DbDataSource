Examples

             You can add a datasource in your markup like so:


             <ss:DbDataSource runat="server" Source="Namespace.MyDbContext.MyDbSetProperty" />


             You can also filter the result with LINQ in the Where property and select values with the Select property like so:


             Where:


             <ss:DbDataSource runat="server" Source="..." Where='<%# (Func<IQueryable, IQueryable>)(set => set.OfType<MyEntityType>().Where(s => My where expression here)) %>' />


             Select:


             <ss:DbDataSource runat="server" Source="..." Select='<%# (Func<IQueryable, IQueryable>)(set => set.OfType<MyEntityType>().Select(s => My select expression here)) %>' />
